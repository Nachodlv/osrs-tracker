# Iron Tracker

Iron Tracker is a visual goal tracker for Old School RuneScape ironman accounts, inspired by [Ladlor's Chart](https://ladlorchart.com/). It lays out gear, quests, diaries, and skill goals as a dependency graph so you can see what unlocks what, track progress per-goal, and reorder or customize the tree to match your own account's plan. Progress is saved locally in your browser, with support for multiple profiles and import/export.

This project is a work in progress, so expect rough edges. It was built with AI assistance for the code generation.
